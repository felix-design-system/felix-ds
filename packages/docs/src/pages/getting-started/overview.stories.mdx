<SubTitle>Introduction</SubTitle>

<Text>Felix DS is an open-source React Design System Library that uses
TailwindCSS and Radix UI to create beautiful, reusable, accessible and
server/client side components for your application.</Text>

<Text>Advantages of using Felix DS:</Text>

<List.Root>
  <List.Item>component standardization;</List.Item>
  <List.Item>easy to use in your application;</List.Item>
  <List.Item>because Felix DS uses TailwindCSS for styles, there is no runtime
  CSS, and the library is compatible with React v18 and server side
  components;</List.Item>
  <List.Item>you may extend TailwindCSS and the default theme configuration in
  your app, if you want to use Tailwind; in other hands, you don't need to
  install and use TailwindCSS.</List.Item>
</List.Root>
