# Installation

Learn how to install and use Felix DS in your app.

## Install package

To install Felix DS, run one of the following commands:

<Tabs.Root defaultValue="npm" data-mode="dark" className="border border-neutral-700">
  <Tabs.List>
    <Tabs.Trigger value="npm">npm</Tabs.Trigger>
    <Tabs.Trigger value="yarn">yarn</Tabs.Trigger>
    <Tabs.Trigger value="pnpm">pnpm</Tabs.Trigger>
  </Tabs.List>
  <Tabs.Content value="npm">
```bash
npm install @felix-ds/react@latest
```
  </Tabs.Content>
  <Tabs.Content value="yarn" asChild>
```bash
yarn add @felix-ds/react@latest
```
  </Tabs.Content>
  <Tabs.Content value="pnpm" asChild>
```bash
pnpm add @felix-ds/react@latest
```
  </Tabs.Content>
</Tabs.Root>

## Import global style

You need to add `global.css` style into your application. If you are using
webpack and CSS, CSS Modules or Saas, import global.css file into your CSS
file:

<Source
  language="css"
  dark
  code={`
@import "@felix-ds/react/dist/global.css";
`}
/>

If you are using Vite or Next.js framework, just import CSS directly into
your main JSX/TSX file:

<Source language="tsx" dark code={`import "@felix-ds/react/dist/global.css"`} />

For other frameworks, see the documentation about how to import CSS.

<SubTitle>Use TailwindCSS</SubTitle>

It's no required to use TailwindCSS in your application, but if you want
to use it, you need to extend the default theme configuration of Felix DS into
your Tailwind configuration file.

To do this, change `tailwindcss.config.ts` with the follow content:

<Source
  language="ts"
  dark
  type="code"
  code={`import { tailwindConfig } from '@felix-ds/react'
\nexport default {
  content: ['./src/**/*.tsx'],
  theme: {
    extend: tailwindConfig.theme,
  },
}
`}
/>

To override all default configuration (recommended):

<Source
  language="ts"
  dark
  type="code"
  code={`import { tailwindConfig } from '@felix-ds/react'
\nexport default {
  content: ['./src/**/*.tsx'],
  theme: tailwindConfig.theme,
}
`}
/>
